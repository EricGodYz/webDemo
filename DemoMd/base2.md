```javascript
1、访问对象成员 : 
		对象.属性   对象.方法() 
		对象[变量或字符串]
2、删除属性:
		delete obj.attr

		}
```

### 数组  

```javascript
1、push()  在数组的后面添加(单个或多个)新元素，返回数组新的长度
2、pop()   删除数组的最后一个元素，返回被删除的元素
3、unshift()  在数组的前面添加(单个或多个)新元素，返回数组新的长度
4、shift()  删除数组的第一个元素，返回被删除的元素
5、splice()  可以对数组进行增、删、改，返回一个数组(被删除的元素)
 		--删除n项 : arr.splice(起始位置,n)
		--增加元素： arr.splice(起始位置,0,添加1，添加2)
		--修改元素： arr.splice(起始位置,2,替换1,替换2)
6、sort()  将数组进行排序，返回数组
	arr.sort()  默认按照字符编码排序
7、reverse() 将数组进行倒转,返回数组
   1-7的操作都会改变原数组
 
  
1、slice() 从数组中拷贝一部分，返回新数组
	arr.slice(1,4) 包含索引为1的元素，不包含索引为4的元素(拷贝索引为 1 2     3 的元素)
2、concat() 用于合并数组，返回新数组
	arr.concat(arr1,arr2......)
3、toString() 将数组转成字符串，返回字符串
4、join() 将数组转成字符串，参数为分隔符，返回字符串
```
